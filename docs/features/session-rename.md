# 会话重命名功能

## 功能说明

会话重命名功能允许用户自定义修改会话的名称，使会话管理更加灵活和个性化。

## 使用步骤

### 1. 找到要重命名的会话

在左侧会话列表中，找到您想要重命名的会话。会话按时间分组显示：
- 今天
- 昨天
- 最近 7 天
- 最近 30 天
- 去年
- 更早

### 2. 打开重命名对话框

将鼠标悬停在会话上，会在右侧显示一个三点菜单图标（⋯）。点击该图标，在弹出的下拉菜单中选择"重命名会话"。

### 3. 输入新名称

在弹出的对话框中：
- 输入框会自动聚焦并选中当前会话名称
- 输入新的会话名称（最多 100 个字符）
- 会话名称不能为空

### 4. 确认修改

有两种方式确认修改：
- 点击"确认"按钮
- 按 Enter 键

如果不想修改，可以点击"取消"按钮或按 Esc 键关闭对话框。

## 功能特性

### 实时更新
重命名成功后，会话列表中的名称会立即更新，无需刷新页面。

### 多语言支持
界面支持中文和英文：
- 中文：重命名会话、请输入会话名称、重命名成功等
- English: Rename Session, Enter session name, Renamed successfully, etc.

### 输入验证
- 会话名称不能为空
- 最大长度限制为 100 个字符
- 如果输入无效，会显示友好的提示消息

### 快捷操作
- 对话框打开时自动聚焦输入框
- 自动选中当前名称，方便快速替换
- 支持 Enter 键快速确认

## 技术实现

### API 接口
```
PUT /api/v1/sessions/:id
Content-Type: application/json

{
  "title": "新的会话名称"
}
```

## 常见问题

### Q: 重命名后会话内容会丢失吗？
A: 不会。重命名只修改会话的显示名称，不会影响会话中的任何消息内容。

### Q: 可以给多个会话设置相同的名称吗？
A: 可以。系统不限制会话名称的唯一性，您可以根据需要自由命名。

### Q: 重命名失败怎么办？
A: 如果重命名失败，系统会显示错误提示。请检查网络连接，然后重试。如果问题持续存在，请联系管理员。

### Q: 可以撤销重命名操作吗？
A: 可以。只需再次打开重命名对话框，输入之前的名称即可。
